# Counting-fingers

Finger Counting AI Walkthrough
The Finger Counting AI is a premium web application that uses MediaPipe's hand tracking model to detect and count fingers in real-time.

Features
Real-time Hand Tracking: Detects up to 2 hands with 21 landmarks each.
Finger Counting Logic: High-accuracy finger counting based on landmark positions.
Glowing Landmarks Overlay: Visual feedback with a stunning neon hand mesh.
Premium Glassmorphism UI: A modern, dark-themed design with blur effects and smooth transitions.
How to Use
Launch the App: Ensure the dev server is running.
Start Camera: Click the Start Camera button.
Grant Permissions: Allow the browser to access your camera when prompted.
Show Your Hand: Position your hand clearly in the video feed.
Count Fingers: Watch the "Fingers Detected" display update as you show different numbers of fingers.
Design Highlights
Dynamic Background: Animated radial gradients for a premium feel.
Responsive Layout: Works on desktop and mobile browsers.
Interactive Feedback: The count display glows and scales when fingers are detected.
